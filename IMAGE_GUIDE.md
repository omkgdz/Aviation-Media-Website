# Image Assets Required

This project requires the following image directories and files. Please add your own aviation photography images to these locations.

## Directory Structure

```
public/images/
├── hero/
│   ├── hero1.jpg      # Hero image 1 (recommended size: 1920x1080)
│   ├── hero2.jpg      # Hero image 2
│   └── hero3.jpg      # Hero image 3
├── members/
│   ├── member1.jpg    # Team member photo (recommended size: 400x400)
│   ├── member2.jpg
│   ├── member3.jpg
│   ├── member4.jpg
│   ├── member5.jpg
│   └── member6.jpg
├── gallery/
│   ├── aircraft1.jpg   # Aircraft photo
│   ├── aircraft2.jpg
│   ├── aircraft3.jpg
│   ├── aircraft4.jpg
│   ├── aircraft5.jpg
│   ├── aircraft6.jpg
│   ├── aircraft7.jpg
│   ├── aircraft8.jpg
│   └── thumb/         # Thumbnail versions (recommended size: 400x300)
│       ├── aircraft1.jpg
│       └── ...
└── reports/
    ├── airshow2024-cover.jpg
    ├── singapore2024-cover.jpg
    ├── beijing2023-cover.jpg
    └── paris2023-cover.jpg
```

## Recommended Image Sizes

- **Hero Images**: 1920x1080 pixels (landscape)
- **Team Member Photos**: 400x400 pixels (square)
- **Gallery Photos**: 1920x1080 pixels (landscape)
- **Gallery Thumbnails**: 400x300 pixels
- **Report Covers**: 1920x1080 pixels (landscape)

## Image Formats

All images should be in JPG or PNG format. WebP format is recommended for better performance.

## Notes

- The images will display with gradient overlays and effects
- You can update the image paths in the data files located in `src/data/`
- Currently, the project uses placeholder paths that you'll need to replace with your actual images
