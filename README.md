# AnDing Spotting

Professional Aviation Photography & Media Website

## Features

- **Home Page**: Hero section with rotating photos, team member profiles with social links
- **Photo Gallery**: Filterable aircraft photography with lightbox viewer
- **Airshow Reports**: Detailed coverage of aviation events
- **Spotting Log**: Airport-based spotting session logs with observations

## Technology Stack

- React 18 + TypeScript
- Vite
- TailwindCSS 3
- React Router DOM v6
- Zustand (State Management)
- Lucide React (Icons)

## Getting Started

### Prerequisites

- Node.js 20+
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/       # Reusable UI components
├── pages/           # Page components
├── data/            # Mock data and types
├── store/           # Zustand state management
└── App.tsx          # Main application
```

## Features Overview

### Team Members
Each team member card includes links to:
- Instagram

### Photo Gallery
- Category filtering (Commercial, Military, General Aviation, Helicopters)
